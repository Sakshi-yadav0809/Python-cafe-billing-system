# Python-cafe-billing-system

A simple Python-based Cafe Management System that allows users to:

- View the cafe menu
- Order items
- Add multiple items to an order
- Calculate the total bill
- Handle unavailable menu items

## Technologies Used

- Python

## Features

- Menu Management using Dictionary
- User Input Handling
- Billing System
- Multiple Item Ordering
- Error Handling for Invalid Items

## Run

```bash
 cafe.py
